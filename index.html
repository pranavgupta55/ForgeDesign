Stats Row

Big serif numerals + small uppercase labels, separated by 1 px subtle vertical rules. Counts below are driven by the live demo state — click action buttons in the section windows to see them update.

Bookended Filter Bar

Select-all / Select-none bookends flank a centered chip row. Click chips to toggle individually; the bookend buttons fully select or clear the set.

Section Windows — live demo

Bordered card with a header strip (status dot · lowercase label · right-aligned count), followed by a mono-set column-header row and data rows. Click the action cluster on any row to move tasks between buckets (active → pending → done → archived). Stats above update automatically.

Stack Pairs

Two-line vertical stacks. Title + ID (Inter title above mono id) and TOC + Time (uppercase tagline above mono numeric).
